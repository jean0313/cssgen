# CSS Generator

a tiny tool using template to generate css files

## Usage
config example:
```json
{
  "tpl": "jt.tmpl",
  "config-datas": [
    {
      "out-file-path": "par.css",
      "data": {
        "mainColor": "red",
        "hoverColor": "#fff"
      }
    },
    {
      "out-file-path": "jcmt.css",
      "data" : {
        "mainColor": "green",
        "hoverColor": "#000"
      }
    }
  ]
}
```
